<h1>Назва застосунку; Проєкт: "Expense Tracker" (Трекер витрат)</h1>

<h2>Мета</h2> 
<br>
Програма допомагає користувачам вести облік витрат, категоризувати їх і переглядати статистику за певний період.<br> Це корисний інструмент для тих, хто хоче контролювати свої фінанси. (які задачі вирішує);

<h3>Користувачі та їх можливості;</h3>

<ol>
<li>Користувач (з реєстрацією)</li>
<li>Додавати витрати</li>
<li>Видаляти та редагувати записи</li>
<li>Переглядати витрати у вигляді списку</li>
<li>Фільтрувати витрати за датою або категорією</li>
</ol>

Основні сутності (основні об'єкти, якими оперує застосунок, та їх поля);
Витрата (Expense)

Категорія (Їжа, Транспорт, Розваги тощо)

Дата

Коментар

Додавання, редагування та видалення витрат

Вибір технології WinForms.
